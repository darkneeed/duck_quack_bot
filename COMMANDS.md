# Команды бота Школы 21 (Волгоград)

## Для пользователей (личка бота)

| Команда | Описание |
|---------|----------|
| `/start` | Начать верификацию или узнать статус заявки |
| `/profile` | Показать профиль на платформе Школы 21 (уровень, XP, коалиция, проекты, навыки) |

### Для администраторов (личка бота)

| Команда | Описание |
|---------|----------|
| `/post` или `/пост` | Опубликовать пост в общий тред чата: сначала текст с HTML-разметкой, затем опциональная картинка |

### Сценарий верификации
1. Пользователь пишет `/start`
2. Вводит логин на платформе Школы 21
3. Бот проверяет логин через API
4. Пользователь опционально добавляет комментарий
5. Заявка уходит на модерацию
6. После одобрения — пользователь получает одноразовую ссылку + правила чата

## Для модераторов (чат модерации)

### Управление заявками
Кнопки на карточке заявки:
- **✅ Одобрить** — одобрить заявку, сгенерировать инвайт-ссылку, уведомить пользователя
- **❌ Отклонить** — отклонить с указанием причины и установкой кулдауна (нет / 1ч / 24ч / 72ч)

### Команды пользователей

| Команда | Описание |
|---------|----------|
| `/ban <tg_id> [причина]` | Заблокировать пользователя |
| `/unban <tg_id>` | Разблокировать пользователя |
| `/deluser <tg_id>` | Удалить пользователя и все его заявки из БД (с подтверждением) |
| `/userinfo <tg_id>` | Показать информацию о пользователе из БД |

### Экспорт и данные

| Команда | Описание |
|---------|----------|
| `/export` | Выгрузить всю БД в CSV-файл |
| `/cleardb` | Полностью очистить БД (с двойным подтверждением) |

### Мероприятия

| Команда | Описание |
|---------|----------|
| `/events` | Показать ближайшие мероприятия кампуса (следующие 30 дней) |

## Автоматические уведомления

| Топик | Содержимое | ENV переменная |
|-------|-----------|----------------|
| Мероприятия | Новые события на платформе (опрос каждые 5 мин) | `EVENTS_TOPIC_ID` |
| Кампус | Кто занял место в кластере (опрос каждые 2 мин) | `WORKSTATION_TOPIC_ID` |
| Новые участники | Карточка нового участника при входе в чат | `NEWCOMER_TOPIC_ID=278` |
| Дайджест | Еженедельная статистика кампуса каждый понедельник в 09:00 UTC | `DIGEST_TOPIC_ID=197` |

---

## ENV переменные

| Переменная | Описание | Пример |
|------------|----------|--------|
| `BOT_TOKEN` | Токен бота от @BotFather | `123456:ABC...` |
| `ADMIN_IDS` | TG ID модераторов через запятую | `123456,789012` |
| `MODERATION_CHAT_ID` | ID чата модерации | `-1001234567890` |
| `MODERATION_TOPIC_ID` | Топик для карточек заявок | `0` |
| `COMMUNITY_CHAT_ID` | ID чата участников | `-1001234567890` |
| `S21_API_USERNAME` | Логин для API Школы 21 | `your-login` |
| `S21_API_PASSWORD` | Пароль для API Школы 21 | `your-password` |
| `S21_CAMPUS_ID` | UUID кампуса | `14b2cc80-...` |
| `DB_PATH` | Путь к SQLite БД | `data/bot.db` |
| `INVITE_LINK_EXPIRE_SECONDS` | Время жизни инвайт-ссылки | `86400` |
| `EVENTS_TOPIC_ID` | Топик для мероприятий | `0` |
| `WORKSTATION_TOPIC_ID` | Топик для уведомлений о кампусе | `272` |
| `NEWCOMER_TOPIC_ID` | Топик для новых участников | `278` |
| `DIGEST_TOPIC_ID` | Топик для еженедельного дайджеста | `197` |

## Права бота

### В чате модерации
- Отправка сообщений
- Отправка документов (для `/export`)

### В чате участников
- Отправка сообщений
- Создание инвайт-ссылок
- **Администратор** с правом `can_manage_tags` — для установки тегов участникам
