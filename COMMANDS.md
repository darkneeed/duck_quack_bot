# Команды бота

> Этот файл синхронизирован с каталогом `s21_bot/command_catalog.py`.

Ниже перечислены реальные команды и алиасы, разбитые по контексту использования.

## Личка бота

| Команда | Алиасы | Кому | Описание | Где работает |
|---------|--------|------|----------|--------------|
| `/start` | — | Участник | главное меню и запуск/повторный старт заявки | private |
| `/changetg` | — | Участник | подсказка по смене Telegram-аккаунта | private |

## Команды участников со scope-настройкой

| Команда | Алиасы | Кому | Описание | Где работает |
|---------|--------|------|----------|--------------|
| `/profile` | — | Участник | показать профиль на платформе | scope: <code>cmd_profile_scope</code> |
| `/where <логин>` | — | Участник | показать, где участник сидит в кампусе | scope: <code>cmd_where_scope</code> |
| `/пир <логин>` | `/peer <логин>` | Участник | показать карточку участника сообщества | scope: <code>cmd_peer_scope</code> |
| `/пиры <проект>` | `/peers <проект>` | Участник | найти участников, которые сейчас делают проект | scope: <code>cmd_peers_scope</code> |
| `/логтайм [логин]` | `/logtime [логин]` | Участник | посмотреть среднее время в кампусе за неделю | scope: <code>cmd_logtime_scope</code> |
| `/топ` | `/top` | Участник | показать топ участников по XP | scope: <code>cmd_top_scope</code> |
| `/вкампусе` | `/incampus` | Участник | показать, кто из участников сейчас в кампусе | scope: <code>cmd_incampus_scope</code> |
| `/мероприятия` | `/events` | Участник | показать ближайшие мероприятия кампуса | scope: <code>cmd_events_scope</code> |

## Примечания

- Для scope-команд доступность задаётся runtime-настройками `CMD_*_SCOPE` через `/admin`.
- Команды модерации в чате сообщества работают только у модераторов/админов.
- Команды в личке бота, требующие статус участника, доступны только после одобрения заявки.
